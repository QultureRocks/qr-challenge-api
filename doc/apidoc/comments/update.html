PATCH /api/v1/comments/:id
Updates a comment

PUT /api/v1/comments/:id
Updates a comment

This endpoint updates an user based on the received data.

Errors

Code Description Metadata
422 Unprocessable Entity - returned when user data is incorrect 
:status: '422'
:error: Unprocessable Entity

Examples

User before update

{
  "comment": {
    "user_id": 3,
    "value": "Bojack is awesome!"
  }
}

Request

{
  "comment": {
    "name": "Bojack is incredible!"
  }
}

Response

{
  "comment": {
    "user_id": 3,
    "name": "Bojack is incredible!",
    "user": {
      "name": "Bojack new name",
      "email": "bojack@example.com.br",
      "job_title": 'Actor',
      "admission_date": '2015-11-01',
      "photo_url": 'https://imgur.com/gallery/something_about_bojack',
      "created_at": "2017-03-15T20:30:13.354-03:00",
      "updated_at": "2018-03-15T20:30:13.354-03:00"
    }
  }
}

Params

Param name Description
comment
required

Comment JSON object

Validations:

  • Must be a Hash
comment[value]
required

The new comment

Validations:

  • Must be a String
