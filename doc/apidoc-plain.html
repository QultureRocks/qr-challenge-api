Comments / create

POST /api/v1/comments
Creates a comment

This endpoint creates a new comment based on the received data.

Examples

Request

{
  "comment": {
    "user_id": 3,
    "value": "Bojack is awesome!"
  }
}

Response

{
  "comment": {
    "id": 3,
    "value": "Bojack is awesome!",
    "user_id": 3,
    "user": {
      "id": 3,
      "name": "Bojack",
      "email": "bojack@example.com.br",
      "job_title": 'Actor',
      "admission_date": '2015-11-01',
      "photo_url": 'https://imgur.com/gallery/something_about_bojack',
      "created_at": "2018-03-15T20:30:13.354-03:00",
      "updated_at": "2018-03-15T20:30:13.354-03:00"
    },
    "created_at": "2018-04-15T20:30:13.354-03:00",
    "updated_at": "2018-04-15T20:30:13.354-03:00"
  }
}

Errors

422 Unprocessable Entity - returned when comment data is incorrect 
:status: '422'
:error: Unprocessable Entity

Params

  • comment : required [

    Must be a Hash

    ]

    Comment JSON object

    • user_id : required [

      Must be a Integer

      ]

      Id of the user to comment

    • value : required [

      Must be a String

      ]

      The actual comment
